## About Metro b00da

This project aims to provide the best path between two train stations.

## Software requirements

| Software        | Version     |
| -----------      | ---------------     |
| git | >= 2.32.0 |
| Node.js | >= v14.17.1 |
| npm | >= 7.19.1 |

## Installation

* `git clone https://github.com/Lmendev/metro-b00da`
* `cd metro-b00da`
* `npm i`
* `npm start`
